1.在node官网https://nodejs.org/en/下载node.js

2.在自己电脑中新建一个文件夹，例如：D:/node

3.在该新建的文件夹中，打开命令行窗口（shift+右击）

4.通过npm init创建package.json文件

通过 npm install express 安装express模块，后面要用到的

5.在node文件夹中新建app.js文件，node app.js运行
